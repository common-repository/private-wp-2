Makes your blog (including RSS feeds) accessible only to logged in users.

== Description ==

Private WP 2 will make sure people can only read your blog and rss feeds after they log in. Already logged in users will see no difference.



== Installation ==

1. Upload `private-wp-2` folder to the `/wp-content/plugins/` directory
2. Activate Private WP 2 through the 'Plugins' menu in WordPress
3. Done!
